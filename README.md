### About

Backend engineer based in Pune, building production-grade microservices and multi-agent AI systems for the hospitality SaaS world. Two-plus years at Upswing Cognitive Hospitality Solutions, where I've architected gRPC services on GKE, wired 6+ Property Management Systems into a unified abstraction, and shipped five iterations of an enterprise conversational AI agent across OpenAI, Gemini, Claude, and Groq. Star Performer of the Quarter (still quietly proud of that one). When I'm not at the terminal, you'll find me climbing a fort in the Sahyadris, sketching, gaming, cooking, or singing along to something embarrassing.

### 🏗️ Featured work

**Multi-Agent Conversational AI System**
An enterprise AI assistant evolved across 5 iterations. Implemented Agentic AI patterns with MCP-based tool calling, versioned prompt templates with context-aware chaining and dynamic composition, and a RAG retrieval layer over ChromaDB. Dynamic model routing across 4 LLM providers (OpenAI, Gemini, Claude, Groq).

**Enterprise Third-Party Integration Platform**
A unified backend integration layer connecting 6+ Property Management Systems, enterprise door lock platforms, and Guest Room Management Systems. Abstract base classes with provider-specific implementations, Redis caching for reservation and availability sync, and webhook event receivers normalizing third-party callbacks onto an internal event bus.

**Self-Hosted CI/CD Infrastructure**
Self-hosted GitHub Actions runner servers executing internal CI/CD pipelines. Automated Docker image builds with semantic versioning, container registry pushes, Trivy vulnerability scanning, and Helm-driven Kubernetes rollouts to GCP staging and production clusters.

**RAG Knowledge Retrieval Pipeline**
An end-to-end Retrieval-Augmented Generation pipeline using Playwright for scraping, BeautifulSoup for parsing, OpenAI embeddings, and ChromaDB as the vector store. Chunking, indexing, and semantic search powering context-aware AI responses.
